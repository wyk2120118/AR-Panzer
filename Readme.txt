Wenn Sie diese Projekt herunterladen, mussen Sie die Code über des Pfads von Panzermodell ändern. (In der 42 Zeile von ar_content.cpp)
Panzermodell ist in "\\ARapp\\tank\\tank.obj"

Tastatur:
F1 wechselt in den Schießmodus
F2 beendet den Schießmodus
F3 vergrößert den Panzer
F4 verkleinert den Panzer
F5 schießt
Forward, Return, Left und Right steuern die Bewegung von Panzer
Page UP, Page DOWN drehen Panzer

Mouse
Linke Maustaste dreht Panzer gegen den Uhrzeigersinn
Rechte Maustaste dreht Panzer im Uhrzeigersinn
